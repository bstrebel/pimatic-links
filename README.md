pimatic links plugin
=======================

Adds links to the main menu in the mobile frontend.

Config:
-------

```json
{
  "plugin": "links",
  "links": [
    {
      "title": "pimatic.org",
      "url": "http://pimatic.org"
    },
    {
      "title": "google.com",
      "url": "http://google.com"
    },       
  ]
}
```
